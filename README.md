# perfect-guess-game
A Python number guessing game where the player tries to guess a randomly generated number with hints for higher or lower guesses

## Features
- Random number generation
- Higher/Lower hints
- Guess counter
- Simple command-line interface

## Technologies Used
- Python 3
- Random Module

## How to Run
1. Clone the repository
2. Open the project folder
3. Run:
   python main.py

## Sample Output
Guess the number: 25
Lower number please

Guess the number: 10
You have guessed the number 10 correctly in 7 attempts.
